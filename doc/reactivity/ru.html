Реактивность

Реактивная переменная - это Proxy, которая обладает следующими характеристиками:

  1. Все свойства/элементы этого объекта являются такм же Proxy

  2. Если изменить или создать нове свойтсво этому объекту другим объектом, то присваиваемый объект станет такм же Proxy

  3. Когда происходит вычисление выражений команд шаблона, учавствующие в этих выражениях реактивные переменные будут связаны с HTML элементов на котором происходит вычисление

  4. При изменение реактивной переменной, связанные с ним HTML элементы будут отправлены на рендер

Все объекты области видимости элементов являются реактивными. Что позволяет реализовывать реактивное взаимодействия без каких либо дополнительных действий.

Для создания реактивного объекта можно воспользоваться функций mw_getProxy(value). Так же есть глобальный рекстивный объект glob.

Есть объекты которые не будут становится реактивной переменной, это станртные объекты:

  • FormData
  • Document
  • DocumentFragment
  • HTMLElement
  • Text
  • Date
  • Request
  • Response

    • Список этиъ объектов может быть в будующем расширен

    Для того что бы сделать объект, который не будет становится реактивной переменной, нужно устновить ему свойства mw_p_target равное null: FormData.prototype[p_target] = null
