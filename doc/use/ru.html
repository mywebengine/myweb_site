Поключение

Что бы подключить myweb нужно добвить скрипт и CSS-файл на страницу. Размер сжатой минимизированной версии (алгоритмом Brotli): скрипт 28 KB и CSS-файла 1 KB.

Подключать CSS-файл не обязательно, будет достаточно, если вы скопируете в свой CSS правило (это правило нужно для скрытия элементов тегом <template> и для скртия элеменов командой filling):

template, *[is_filling] > * { display: none !important; }

Файлы можно взять с сайта https://mywebengine.org

<script src="https://mywebengine.org/myweb/myweb.js" type="module" async=""> <link href="https://mywebengine.org/myweb/myweb.css" type="text/css" rel="stylesheet" />

Подключение минимизированнй версии:

<script src="https://mywebengine.org/myweb/myweb.min.js" type="module" async=""> <link href="https://mywebengine.org/myweb/myweb.min.css" type="text/css" rel="stylesheet" />

Файлы можно взять с GitHub

git clone git@github.com:mywebengine/myweb.git

Подключение файлов с вашего хоста:

<script src="/myweb/myweb.js" type="module" async=""></script> <link href="/myweb/myweb.css" type="text/css" rel="stylesheet" />

Подключиние минимизированной версии с вашего хоста:

<script src="/myweb/myweb.min.js" type="module" async=""></script> <link href="/myweb/myweb.min.css" type="text/css" rel="stylesheet" />

Параметры подключения

Подключение скрипта можно конфигурировать используя GET параметры:

<script src="/myweb/myweb.min.js?debug=(0|1|2)&load=(onload|skip)&linking=1" type="module" async=""></script>
  • debug

    Инициализация переменной mw_debugLevel уровеня режима отладки

  • load

    Если указать onload - то первый проход рендера по документу будет запущен после полной загрузки страницы.

    Если указать skip - то первый проход рендера будет отменён.

  • linking

    Если указать значение "1", то первый проход рендера будет в режиме линоковки, подробнее это обсуждается в разделе серверный рендеринг, режим линковки
