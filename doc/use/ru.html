Поключение

Что бы подключить myweb нужно добвить скрипт и CSS-файл на страницу. Размер сжатой минимизированной версии скрипта 28 KB, размер CSS-файла 1 KB.

Подключать CSS-файл не обязательно, будет достаточно, если вы скопируете в свой CSS правило (это правило нужно для скрытия элементов тегом <template>):

template, *[is_filling] > * { display: none !important; }

Файлы можно взять с CDN

<script src="https://cdn.jsdelivr.net/gh/mywebengine/myweb@latest/myweb.min.js" type="module" async=""></script> <link href="https://cdn.jsdelivr.net/gh/mywebengine/myweb@latest/myweb.min.css" type="text/css" rel="stylesheet" />

Файлы можно взять с GitHub

git clone git@github.com:mywebengine/myweb.git

И подулючить минимизированные версии:

<script src="/myweb.min.js" type="module" async=""></script> <link href="/myweb.min.css" type="text/css" rel="stylesheet" />

Или подключить основные файлы:

<script src="/myweb.js" type="module" async=""></script> <link href="/myweb.css" type="text/css" rel="stylesheet" />

Параметры подключения

Подключение скрипта можно конфигурировать используя GET параметры:

<script src="/myweb.min.js?debug=(0|1|2)&load=(onload|skip)&linking=1" type="module" async=""></script>
  • debug

    Инициализация переменной mw_debugLevel уровеня режима отладки

  • load

    Если указать onload - то первый проход рендера по документу будет запущен после полной загрузки страницы.

    Если указать skip - то первый проход рендера будет отменён.

  • linking

    Если укаать значение "1", то первый проход рендера будет в режиме линоковки, подробнее это обсуждается в разделе серверный рендеринг, режим линковки
