fetch - загрузка данных

Команда позволяет загрузить данные по сети

Синтаксис

<div fetch="expression">...</div>

Описание

Возвращаемое значение выражения может быть черех типов:

  • string

    Строка будет интерпретирована как URL

  • Request (стандартный объект)

    Этот объект будет использоваться функцией fetch для загрузки данных

  • Response (стандартный объект)

    Этот объект будет использоваться как результат работы команды. Применение команды в таком варианте может пригодится при тестировании кода.

  • == false

    Если значение можно интерпретировать как ложь, тогда рендер продолжется со следующего тега (дети рендерится не будут)

Процесс получение данных - события

В процессе получения данных с сервера, на текущий элемени отправляются следующие события:

  • load

    Отправляется после получения данных. Ответ сервера находится в: <текущее событие>.detail.res (это Response).

  • ok

    Отправляется после получения данных, если статус ответа сервера равен 200: <текущее событие>.detail.res.ok === true. Ответ сервера находится в: <текущее событие>.detail.res (это Response).

  • error

    Отправляется если в процессе загрузки произошла ошибка. Объект произошедшей ошибки находится в: <текущее событие>.detail.err.
