watch - отслеживание изменений

Команда позволяет отслеживать изменения в рективых объектах. Выполняет выражение как js-код, в контексте текущего элемента.

Синтаксис

<div watch[.var-name]="expression">...</div>

Параметры

  • var-name

    Имя переменной в которую будет записано значение выражения1

Описание

Отслеживаение изменений работает по принципу: если какая-то рективная переменная, учавствующая в выражении, изменила своё значение (была изменена, перезаписана или удалена) - элемент будет перерендерен.

Пояснения для данного примера
Все присваения запиываются через переменную области видимости w, так как у нас в области видимости есть две ссылки: w - ссылка на область видимости и obj - ссылка на объект с которым мы работаем. И если мы сделаем obj = {...} мы изменим значение переменной, но w.obj останется старое значение.
