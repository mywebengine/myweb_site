scope - область видимости

Команда позволяет поместить область видимости текущего элемента в переменную, имя которой указано в первом параметре. Выражение в данной команде не вычисляется.

Синтаксис

<div scope.var-name="">...</div>

Параметры

  • var-name

    Имя переменной в которую будет записана область видимости текущего элемента1
