html - разметка и текст

Команда создания HTML-разметки

Синтаксис

<div html[.text]="expression"></div> <div>${expression}</div>

Значение выражения интерпретируется как строка, по которой создаётся разметка через .innerHTML или .textContent.

Параметры

  • text необязательный

    Eсли параметр указан, то значение выражения интерпретируется как текст: .textContent = expression. Название параметра может быть любым.

  • ${expression}

    Эта кнострукция эквивалентна вызову команды с первым параметром - значение будет интерпретировано как текст.
