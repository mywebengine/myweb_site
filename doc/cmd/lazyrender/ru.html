lazyrender - ленивый рендеринг

Этот атрибут позволяет включить режим ленивого рендеринга на элементе у которого он установлен.

Ленивый рендер - это алгоритм при котором рендерятся только элементы в области видимости, а то что не видно, ожидает того момента когда они попадут в область видимости.

Область видимости - это область текущего экрана + еще две ширины влево и вправо, и две высоты вверх и вниз - это значение по умолчанию, которое записано в файле config.js.

Атрибут проверяется при каждом новом ренедере и его можно можно добавлять или удалять, если того требудет логика приложения.

Синтаксис

<div lazyrender="">...</div>

Если элемент с ленивым рендером "спрятан" за display: none (например, команда filling так дклает с дочерними элементами), то такой элемент считается видимым, так как нельзя с увериностью сказать как нужно обработать такую ситуацию.

Пример

Список в примере отрендерен только на три экрана (как описывалось выше). При скроллировании список рендерится дальше - это можете заметить по размеру движка сколлбара, если открыть пример в отдельной владке.
