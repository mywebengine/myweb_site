Команды

Команда шаблона - это атрибут HTML элемента, который интерпретируется алогоритмом рендера. Любая команда имеет вид:

<div cmd-name[.param.param2.param3 ...]="expression" ...

Каждый HTML элемент может иметь сколь угодно команд, которые интерпертируются последовательно слева-направо.

  • cmd-name

    Имя команды. Далее в разделе документации описываются все команды.

  • param необязательный

    Параметр команды.

    Команда может иметь неограниченное количество параметров, которые отделяются друг от друга символом "."

    Каждая команда интерпретирует параметы по свойму.

    Название параметра может быть пустой строкой, выглядит это так: <div cmd-name.param..param3.="expression" ... Таким образом можно не указывать параметр или добиваться уникальности атрибутов на элементе (название атрибутов у элемента должны быть уникальными).

  • expression

    Выраженеие команды, которое является строкой javascript-кода и выполняется в контексте текущего HTML элемента

Скрытие элементов за тегом <template>

Когда по логике команды она должна скрыть элемент в документе, используетмся механизм переноса DOM элемента в элемент <template>. Для того что бы скрыть элемент на тег <template> применяется CSS правило (которое надожится в файле myweb.css - подробнее о подключении файла можно прочитать в разделе подключение):

template { display: none !important; }

Это правило может инициировать конфиликт с ранее созданой HTML-разметкой, если в нём использубтся теги <template>
