attr

Команда управляет HTML-атрибутами, выполняет выражение, как js-код в контексте текущего элемента.

Синтаксис

<div attr.name[.value-part-or-href-mode]="expression">...

Параметры

  • name - имя атрибута

  • value-part-or-href-mode необязательный - параметр может интерпретироваться двумя способами:

    1. Как часть значения атрибута, которое будет добавлено через пробел, в случаи положительного значенния выражения, или будет удалено, если занчение выражения ложно.

    2. Для атрибута href тага <a> занчение второго параметра может быть push или replace, что будет интерпретировано как создание атрибута со значением значения выражения, при клике на элемент будет вызван метод history.pushState(undefined, undefined, .href) для push или history.replaceState(...) для replace.
