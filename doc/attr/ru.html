attr - атрибуты

Команда управляет HTML-атрибутами. Выполняет выражение, как js-код в контексте текущего элемента.

Синтаксис

<div attr.name[.value-part-or-href-mode]="expression">...

Параметры

  • name

    Имя атрибута

  • value-part-or-href-mode необязательный

    Параметр может интерпретироваться двумя способами:

    1. Как часть значения атрибута, которое будет добавлено через пробел, в случаи положительного значенния выражения,или будет удалено, если занчение выражения ложно.

    2. Для атрибута href тага <a> занчение второго параметра может быть push или replace, что будет интерпретировано как создание атрибута со значением значения выражения, а при клике на элемент событие будет отменено, и будет вызван метод history.pushState(undefined, undefined, .href) для push или history.replaceState(...) для replace.
