if, switch - ветвления

if

Команда if условного ветвлени, значения выражений выполняется, как js-код в контексте текущего элемента.

Может состоять из одного блока if, нескольких блоков elseif и одного else.

Если значение выражение будет интерпретировано как ложное, то тег будет скрыт за тегом <template>.

Синтаксис

<div if[.value-name]="expression">...</div> <div elseif[.value-name]="expression2">...</div> <div elseif[.value-name]="expression3">...</div> <div else="">...</div>

Параметры

  • value-name необязательный

    Имя переменной в которую будет записано занчение выражения, если оно истенно1

switch

Команда switch условного ветвлени, значение выражения выполняется, как js-код в контексте текущего элемента.

Может сосотять из одного блока switch и несколькоих блоков case и одного блока default.

Если значение выражение будет интерпретировано как ложное, то тег будет скрыт за тегом <template>.

Синтаксис

<div switch="expression" case[.value-name]="value1">...</div> <div case[.value-name]="value1">...</div> <div case[.value-name]="value3">...</div> <div default="">...</div>

Параметры

  • value-name необязательный

    Имя переменной в которую будет записано занчение выражения, если оно совподает с переданным в switch1
