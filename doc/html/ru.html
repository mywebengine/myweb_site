html - разметка и текст

Команда создания HTML-разметки, выполняет выражение как js-код, в контексте текущего элемента.

Синтаксис

<div html[.text]="expression">...

Значение выражения интерпретируется как строка, по которой создаётся разметка через .innerHTML или .textContent.

Параметры

  • text необязательный

    Eсли параметр указан, то значение выражения интерпретируется как текст: .textContent = expression. Название параметра может быть любым.

Команда может быть записана через шаблон ${expression}, эта кнострукция эквивалентна вызову команды с первым параметром - значение будет интерпретировано как текст.
