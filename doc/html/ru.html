html

Команда создания HTML-разметки, выполняет выражение как js-код, в контексте текущего элемента.

Синтаксис

<div html[.text]="expression">...

Значение выражения интерпретируется как строка, по которой создаётся разметка через .innerHTML или .textContent.

Параметры

  • text необязательный - если параметр указан, то значение выражения интерпретируется как текст: .textContent = значение выражения. Название параметра может быть любым.

Команда может быть записана через шаблон ${выражение}, эта кнострукция эквивалентна вызову команды с первым параметром - значение будет интерпретировано как текст.
