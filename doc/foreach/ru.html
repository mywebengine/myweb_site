foreach - цикл

Команда повторения HTML-разметки на основывании значения выражения, которое выполняется, как js-код в контексте текущего элемента.

Если значение выражение будет интерпретировано как ложное, то тег будет скрыт за тегом <template>.

Синтаксис

<div foreach[.value-name.key-name]="expression">...</div>

Параметры

  • value-name необязательный

    Имя переменной в которую будет записано занчение текущей итерации1

  • key-name необязательный

    Имя переменной в которую будет записано занчение индекса текущей интерации1
