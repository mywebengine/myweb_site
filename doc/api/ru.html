API

Все функции и объекты находятся в глобальной области видимости self

  • glob

    Глобальный реактивный объект

  • mw_$src

    Родителький DOM элменет для области рендера, равен document.documentElement

  • mw_render($src = mw_$src, delay, scope, isLinking = false)

    Функция запуска рендера

    • $src

      DOM элемент, который будет рендерится

    • delay необязательный

      Задержка в мс перед рендером

    • scope необязательный

      Объект которым будет иницализирован скоп

    • isLinking необязательный

      Признак того что рендере будет в режиме линковки

  • mw_curRender

    Глобальный промис рендера - доступен всегда

  • mw_setDelay(delay)

    Функция устанавлиевает задержку в мс, перед запуском рендера

  • mw_debugLevel

    Уровень режима отладки, может быть устанавлен при подключении скрипта

    • 0

      Отладка выключина

    • 1

      Отладка включена: будут выводится подробные сообщения об ошибках, с указанием в каком файле она произошла

    • 2

      Отладка включена: будут выводится подробные сообщения об ошибках и логирование изменений в реактивных переменных

  • mw_removeChild($src)

    Удаляет DOM элемент $src, очищая данные которые были с неим связаны в процессе рендера

  • mw_incCache

    Кэш вставок - это Map клячами которого являются URL вставок. Применение обсуждается в статье inc.

  • mw_dispatchCustomEvent($src, evtName[, detail])

    Функция для отправки пользовательских событи

    • $src

      Указатель на DOM элемент

    • evtName

      Имя события

    • detail необязательный

      Объект с пользовательскими данными, которые будут доступны при получении события - CustomEvent.detail

  • mw_getProxy(value)

    Функция возвращает реактивный объект (Proxy) на объект value

  • mw_p_target

    Символ - имя свойства для реaктивноого объекта. Значение свойства будет базовым объект на основоании которого был создан реактивный: reactivityObject[mw_p_target]

  • mw_setLoc(url)

    Функция изменяет значение реактивного объекта маршрутизации loc исходя из значения передоваемого в url

  • mw_getUrl(url[, topUrl])

    Функция вернёт нормализованный URL

    • url

      Относительный или абсолютный URL

    • topUrl необязательный

      Функция вернет адрес исходя из того что url указан относительно topUrl

  • mw_p_topUrl

    Символ - имя свойства для DOM элемента. Который предназначен указывать на URL копонента - это нужно в случаи когда пользовательский коптонент находится в другом месте DOM, относительноь своего первоночально положения: $src[mw_p_topUrl].

  • mw_oset(target, propName, newValue)

    Изменяет или создаёт свойство propName объекта target значением newValue. Если target.propName - это объект, массив, Map или Set - то обходит их рекурсивно и изменяет или добавляет значения.

    Смысл этой функции в том, что если объект назначения реaктивный, то изменить в нём только то что изменилось, для того что бы минимизировать количество рендеров.

  • mw_del(target, propName)

    Удаляет из объекта target свойство propName и возвращает значение этого свойства значение

  • mw_copyToClipboard(str)

    Помещает в буфер объмена строку str
