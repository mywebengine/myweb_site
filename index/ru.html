Главная страница myweb

Система myweb - это рассширение функций языка разметки HTML, в попытке превратить его в язык прогораммирования. Для того что бы сделать разработку сайтов понятной и простой.

Этот шаблон обладает следующими качествами:

  1. Простота в использовнии, для использования не нужен API

  2. Ориентирован на HTML

  3. Не блокирующий интерфейс алгоритм рендеринга

  4. Удобный серверный ренденриг (в будующем)
